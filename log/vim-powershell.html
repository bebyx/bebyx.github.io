Як ужиткувати Vim'ом у Windows PowerShell

21-09-2020

Вінда масдай — і малому зрозуміло.

Але поки вона не дай, часом обставини примушують адаптувати прекрасне Лінуксове середовище під потворну, недоладну, незугарну Шінду. Наприклад, на роботі.

Сам стикнувся з ситуацією. І виникла проблема, найбанальніша для GNU/Linux, але, як виявилося, дуже тяжка для Windows 10.

А саме — працювати з текстами через термінал, ясна річ, найліпшим текстовим редактором Vim.

Навіть коли встановлюєш віндо́вський gVim, то ніфіга не працює в PowerShell.

Поліз в Інтернет — самі костурі й неінтуїтивні рішення.

А бавлячися з Terraform'ом, натрапив на такий цікавий менджер пакунків Chocolatey. Так-так, для вінди.

Отам і виявив найпростіший і найзручніший спосіб встановити Vim у PowerShell.

  1. Встановлюєте Chocolatey за інструкцією.
  2. Встановлюєте Vim менеджером пакунків: choco install vim. Все як у людей.
  3. Працюєте з Vim'ом як звичне: приміром, vim file.txt у PowerShell.
  4. Насолоджуєтеся тим, що Vim тепер працює як годиться.

У разі треба би було оновити чи видалити Vim, то робите це теж через Chocolatey, відповідно:

choco upgrade vim

choco uninstall vim

Довідка по командах: choco -h, або choco --help, або choco -?.

Геть альтернативний варіянт

Встановити Git-Bash разом з Git for Windows — і насолоджуватися вшитим у нього з коробки Vim'ом.
